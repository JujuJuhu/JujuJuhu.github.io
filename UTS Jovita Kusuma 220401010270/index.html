Bahasa Dasar Pemograman Web: HTML

Posted by Jovita Kusuma on Friday, 26 May 2023

Semua website moden dan aplikasi web dibuat menggunakan tiga fundamental technologies: HTML, CSS and JavaScript. Ini adalah bahasa pemograman web

Dalam post ini, mari kita fokus ke HTML. Kita akan mempelajari apa itu HTML dan mengapa kamu harus mempelajarinya

Apa itu HTML

HTML merupakan singkatan dari HyperText Markup Language. Yang merupakan markup language yang dipakai web developer untuk menyusun struktur dan menjelaskan isi konten dari sebuah halaman web (bukanlah sebuah bahasa pemograman)

HTML terdiri dari element-element yang menjelaskan tipe-tipe content yang berbeda: paragraphs, links, headings, images, video, etc. Web browsers menerjemahkan HTML dan merender HTML code sebagai websites.

Pada HTML, setiap element terdiri atas 3 bagian:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at the MDN Web Docs

Mengapa Kamu Wajib Belajar HTML

Ada banyak alasan mengapa kamu wajib belajar HTML, beberapa di antaranya adalah:

  • Supaya dapat menggunakan bahasa pemograman web yang fundamental
  • Agar dapat membuat website yang indah secara mandiri daripada tergantung pada tools seperti Wordpress atau Wix
